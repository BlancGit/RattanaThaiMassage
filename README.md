# Rattana Thai Massage Website

A luxury, single-page website for Rattana Thai Massage in Adelsdorf.

## Project Structure

- `index.html`: Main website structure and content.
- `styles.css`: Custom CSS variables and styles for the "Thai Oriental Luxury" theme.
- `assets/`: Directory for images and media.

## Development

To run locally:

```bash
npx http-server . -p 8080
```

## Deployment

This project is ready for deployment on Vercel.

1. Import this repository into Vercel.
2. Vercel will automatically detect the static project.
3. Deploy!

### Configuration

A `vercel.json` file is included to configure:
- Clean URLs
- Trailing slashes behavior
